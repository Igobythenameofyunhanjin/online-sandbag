## Running Locally

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/online-sandbag.git
```

--

### 2. Install dependencies

For frontend:

```bash
cd frontend
npm install
``` 

For backend:

```bash
cd backend
npm install
```

--

### 3. Run the project

In two terminals:

```bash
cd frontend
npm run dev
```
```bash
cd backend
npm run dev
```
